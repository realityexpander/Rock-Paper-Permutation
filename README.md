# Rock-Paper-Permutation
Rock Paper Permutation


Given a number of rounds n, return all the possible rock-paper-scissors play possibilities for that number of rounds.
<br>
Examples<br>
Input	Output<br>
roundCount: 1<br>
<br>
[ "r", "p", "s" ]<br>
<br>
roundCount: 2<br>
[ "rr", "rp", "rs", "pr", "pp", "ps", "sr", "sp", "ss" ]<br>
<br>
roundCount: 0<br>
[ ]
